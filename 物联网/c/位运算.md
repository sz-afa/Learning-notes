# 运算符
|运算符 | 功能说明 | 举例 |
| :----: | :----: | :----: |
| ~ | 按位取反  | ~a  |
| & | 按位取与  | a&b  |
| \| | 按位取或  | a&b  |
| ^ | 按位取异或(相同为0不同为1) | a&b  |
| >> | 数据位右移(左边补符号位)  | a>>1  |
| << | 数据位左移(右边补0)  | a<<2  |

# 例子
## 1.将第9位变成1，不要改变其他位置
```c
#include <stdio.h>

int main(void)
{
    /*
        利用将第九位变成1
        00000000 00000101
    |   00000001 00000000  
    
    */
    short a = 5;
    a = a | (1<<8);
    printf("%d\n",a);
    return 0;
}
```
## 2.将第11位变成0，不可以改变其他位置
```c
#include <stdio.h>

int main(void)
{
    /*
        利用将第11位变成0
        00000000 00000101
    &   11111011 11111111  
    */
    short a = 5;
    a = a & ~(1<<10);
    printf("%d\n",a);
    return 0;
}
```
