## 数据读取类型说明
### 脏读:
事务没提交，提前读取
### 不可重复读:
两次读取的数据不一致
![](./img/img7.png)
### 幻读:
事务不是独立执行时发生的一种非预期现象。
![](./img/img8.png)
`事务1将人员表数据性别全部修改为男的过程中，事务2添加一条新数据性别为女。事务1结束时，结果与预期不符合，这就是幻读。`
